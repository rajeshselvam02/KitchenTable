# KitchenTable

A starter **cloud‑kitchen admin UI** built with **Next.js** (TypeScript) that includes:

- **Analytics Dashboard** – revenue, prep‑time, waste charts.
- **Menu Builder** – weekly calendar to assign dishes to days.
- **Order Queue** – real‑time order feed powered by a WebSocket bridge.

The repository contains only the front‑end skeleton; you can connect it to any back‑end (e.g., your existing PropAgent services) by updating the API URLs in `src/lib/api.ts`.
